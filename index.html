Reference Formulas — ASABE S296.5 / ISO 11684 / Bekker Soil-Vehicle Mechanics

Total tractive resistanceF_total = F_roll + F_grade + F_drawbar + F_slip + F_sinkage
Soil thrust (Bekker)T_soil = W · Ks · (1 − e^(−slip/K)) where K = slip coeff.
Rolling resistance (ASABE)F_roll = Cr · W · cos(θ) + b · z^(3/2) · L (b=soil deformation coeff)
Motor torque at wheelτ_motor = (F_total · r) / (GR · η_dt)
Actual ground speedv_actual = v_theoretical · (1 − slip/100)
Ground contact pressureP_contact = W / (n · L · b_track)
Drawbar efficiencyη_db = F_drawbar · v / P_motor · 100%
Traction coefficientμ_traction = F_drawbar / W_driven
